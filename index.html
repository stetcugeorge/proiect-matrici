Problema

Enunțul problemei

Se consideră o matrice cu n linii și m coloane, unde fiecare linie reprezintă o trupă muzicală, iar fiecare coloană reprezintă un album al trupei. Elementul a[i][j] reprezintă numărul de melodii de pe albumul j al trupei i.


Programul citește datele dintr-un fișier text (muzica.in) și scrie rezultatele într-un fișier de ieșire (muzica.out), rezolvând urmotoarele cerințe:


  1. C1. Numărul de albume care conțin mai mult de k melodii
  2. C2. Trupa cu cel mai mare număr total de melodii (maximul pe linii)
  3. C3. Media aritmetică a melodiilor pentru fiecare trupă
  4. C4. Albumele cu număr impar de melodii (care sunt acestea)
Implementare

Cele 4 cerințe

🔢

Numărare condiționată

Se numără toate elementele din matrice care sunt strict mai mari decât un prag k introdus de utilizator. Parcurgere completă cu contor.

🏆

Maxim pe linii

Se calculează suma melodiilor pentru fiecare trupă (sumă pe linie), apoi se identifică trupa cu suma maximă și se afișează indexul acesteia.

📊

Medie aritmetică

Pentru fiecare trupă (linie), se calculează media aritmetică a numărului de melodii de pe toate albumele sale. Rezultat cu două zecimale.

🎵

Filtrare după paritate

Se identifică și se afișează pozițiile (i, j) ale tuturor albumelor cu număr impar de melodii, împreună cu valoarea respectivă.

Soluție

Codul sursă C++

main.cpp
// MusicMatrix — Proiect Matrici | Muzica
// Cerinte: C1 numărare, C2 maxim linii, C3 medii, C4 paritate

#include <iostream>
#include <fstream>
#include <iomanip>
using namespace std;

const int NMAX = 50;

int main() {
    ifstream fin("muzica.in");
    ofstream fout("muzica.out");

    int n, m, k;
    fin >> n >> m >> k;

    int a[NMAX][NMAX];
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            fin >> a[i][j];

    // ——— C1: Albume cu mai mult de k melodii ———
    int contor = 0;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            if (a[i][j] > k) contor++;

    fout << "C1: Albume cu mai mult de " << k << " melodii: "
         << contor << "\n\n";

    // ——— C2: Trupa cu cel mai mare total de melodii ———
    int maxSum = -1, idxMax = 0;
    for (int i = 0; i < n; i++) {
        int suma = 0;
        for (int j = 0; j < m; j++)
            suma += a[i][j];
        if (suma > maxSum) {
            maxSum = suma;
            idxMax = i + 1;
        }
    }
    fout << "C2: Trupa cu cele mai multe melodii: Trupa "
         << idxMax << " (total: " << maxSum << ")\n\n";

    // ——— C3: Media melodiilor pentru fiecare trupă ———
    fout << "C3: Medii aritmetice pe trupe:\n";
    for (int i = 0; i < n; i++) {
        double suma = 0;
        for (int j = 0; j < m; j++)
            suma += a[i][j];
        fout << "  Trupa " << i+1 << ": "
             << fixed << setprecision(2) << suma / m << " melodii/album\n";
    }
    fout << "\n";

    // ——— C4: Albume cu număr impar de melodii ———
    fout << "C4: Albume cu numar impar de melodii:\n";
    bool gasit = false;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            if (a[i][j] % 2 != 0) {
                fout << "  Trupa " << i+1 << ", Album "
                     << j+1 << ": " << a[i][j] << " melodii\n";
                gasit = true;
            }
    if (!gasit)
        fout << "  Niciun album cu numar impar de melodii.\n";

    fin.close();
    fout.close();
    return 0;
}
